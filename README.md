# IP address extensions

This .NET library contains various extensions for the `IPAddress` class.

A NuGet package is available at https://www.nuget.org/packages/Albireo.IPAddressExtensions/

## Requirements

- .NET Framework 4.0
- Visual Studio 2013
- Code contracts
